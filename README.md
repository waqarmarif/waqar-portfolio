# Muhammad Waqar — Portfolio

Personal portfolio website of **Muhammad Waqar**, WordPress Developer & Web Developer at Forbes Middle East. Built with React + Vite.

## Live Demo

> Deploy kar ke link yahan add karo

## Tech Stack

- **React.js** — UI framework
- **Vite** — Build tool
- **CSS3** — Custom styling
- **Font Awesome** — Icons
- **Google Fonts** — Poppins

## Sections

- Hero — Introduction with animated tags
- About — Personal info & details
- Skills — Frontend, Backend, WordPress with progress bars
- Experience — Work history timeline
- Projects — Forbes Events, News Blog, HMS, Tic Tac Toe
- Education — BSIT, FSc, Matric
- Contact — Contact form + info

## Run Locally

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deploy on Vercel

1. Push this repo to GitHub
2. Go to vercel.com and click **Add New Project**
3. Import this repository
4. Framework preset: **Vite**
5. Click **Deploy**
